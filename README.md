## 欢迎使用橙子草的Python自动化脚本

以下是一些功能8 常用键的名称，更多的键名称可以参考 `pyautogui` 的官方文档：

- 字母键：`'a'`, `'b'`, `'c'`, ... , `'z'`
- 数字键：`'0'`, `'1'`, `'2'`, ... , `'9'`
- 功能键：`'f1'`, `'f2'`, ... , `'f12'`
- 箭头键：`'left'`, `'right'`, `'up'`, `'down'`
- 控制键：`'ctrl'`, `'alt'`, `'shift'`, `'win'`, `'cmd'` (Mac OS)
- 其他键：`'enter'`, `'space'`, `'backspace'`, `'tab'`, `'esc'`, `'delete'`, `'home'`, `'end'`, `'pageup'`, `'pagedown'`
